Open the project directory:
cd Sign_In_page

Install all the required dependencies using npm install command.
Install tailwind css.
Change directory to frontend folder:
cd frontend

Run: npm install
 run: npm run dev


Open a new terminal window and change directory to backend folder:
cd server
run npm install
change directory to frontend and run: npm run dev

Open terminal again, change directory to server
run: npm install 
run: npm start


The page uses MySQL database with XAMPP server.
To use it you need to have XAMPP installed on your computer.
The project has .env file which contains the credentials for connecting to the database. You can create one yourself or contact me if you want to use my credentials.
